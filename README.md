# LUIZ. | Growth Estrategista

Landing page profissional para estrategista de growth especializado em odontologia.

## Stack

- **HTML5** — Semântico e acessível
- **CSS3** — Variáveis CSS, Grid, Flexbox, animações
- **Vanilla JS** — Sem frameworks, leve e performático
- **Google Fonts** — Inter

## Estrutura de Pastas

```
Luiz_Growth V2/
├── index.html              # Estrutura semântica principal
├── css/
│   └── main.css            # Estilos completos
├── js/
│   └── main.js             # Interatividade e navegação
├── assets/
│   ├── images/             # Imagens do projeto
│   ├── icons/              # SVGs e favicon
│   └── fonts/              # Fontes locais (futuro)
├── README.md               # Documentação
├── .editorconfig           # Padronização de código
└── .gitignore              # Arquivos ignorados pelo Git
```

## Como Rodar Localmente

1. Clone ou baixe o projeto
2. Abra `index.html` no navegador (ou use uma extensão como Live Server no VS Code)
3. O site carregará automaticamente os arquivos CSS e JS externos

## Funcionalidades

- **Navegação suave** entre seções (smooth scroll)
- **Menu ativo dinâmico** baseado na seção visível
